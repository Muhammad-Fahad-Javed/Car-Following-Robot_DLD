
# 🚗 Car Following Robot
A Digital Logic Design (DLD) laboratory project that implements an autonomous car-following robot capable of detecting, tracking, and following a target vehicle using sensor-based navigation and intelligent control logic.


## 📖 Overview
The Car Following Robot is designed to automatically follow a target vehicle while maintaining a safe distance. The system uses sensors to continuously monitor the position of the object ahead and adjusts its movement accordingly. This project demonstrates the practical application of digital logic, embedded systems, and robotics in automation.


## ✨ Features
- Real-time vehicle detection and tracking
- Automatic speed and direction control
- Safe distance maintenance
- Autonomous movement and navigation
- Sensor-based decision making
- Compact and efficient robotic design
- Practical implementation of Digital Logic Design concepts

## ⚙️ Working Principle
1. The ultrasonic sensor continuously measures the distance to the target vehicle.
2. The microcontroller processes the sensor data.
3. If the target moves forward, the robot follows.
4. If the target slows down or stops, the robot adjusts its speed or stops accordingly.
5. The system maintains a safe and consistent following distance.
